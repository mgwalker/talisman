public void modExp(char op, int num)

Description

Modifies a character's strength or craft experience, depending on the value of op. If op is 's' then change the character's strength by the value in num. If op is 'c' then change the character's craft.

Parameters

char op - What type of operation to perform
int num - Amount to change

Returns

Nothing
